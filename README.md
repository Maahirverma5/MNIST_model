MNIST Real-Time Digit Classifier
A lightweight Deep Learning project that trains a Neural Network to recognize handwritten digits (0-9). It features a Smart Preprocessing pipeline that allows you to write a digit on paper and have it recognized in real-time via webcam.

Quick Start
1. Installation
Clone the repo and install the required libraries:
pip install tensorflow numpy opencv-python matplotlib

2. Train the Model
Run the training script to download the MNIST dataset and build the model. This will generate the mnist_model.keras file.


Accuracy: 97-98%
